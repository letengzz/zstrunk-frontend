# HOWO Tractor Truck

## Product Overview

The **HOWO Tractor Truck** is designed for efficient long-haul transportation. Built on the proven HOWO 7 platform, it delivers reliable performance and cost-effective operation for various hauling applications.

## Key Features

- **Proven Platform**: HOWO 7 series
- **Powerful Engine**: WEICHAI customized
- **Comfortable Cab**: Spacious sleeper
- **Fuel Efficient**: Optimized consumption
- **Easy Maintenance**: Accessible components

## Technical Specifications

| Item | Parameter |
|------|-----------|
| Model | HOWO 7 |
| Engine | WEICHAI WD615/WP10 |
| Power | 336-371 HP |
| Torque | 1380-1500 N·m |
| Transmission | FAST 10/12 speed |
| Drive Form | 4×2 / 6×4 |
| GCW | Up to 60 tons |

## Standard Configuration

### Cab
- HOWO 7 cab with sleeper
- Air suspension seat
- Air conditioning
- AM/FM radio
- Central locking

### Chassis
- STEYR/HANDE axles
- Leaf spring suspension
- Front disc/rear drum brakes
- Fuel tank: 400L

### Transmission
- FAST 10J120T (10 speed)
- Double overdrive
- Optional 12 speed

## Optional Configurations

- 6×4 drive form
- Air suspension
- 600L fuel tank
- ESP system
- GPS navigation

## Applications

- Container transport
- Long-haul freight
- Bulk cargo
- Heavy haulage
- Regional distribution
