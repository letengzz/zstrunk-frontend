# Shacman Tractor Truck

## Product Overview

The **Shacman Tractor Truck** is designed for heavy-duty long-haul transportation. Available in 4×2, 6×4, and 6×6 configurations, it offers reliable performance for various trailer combinations.

## Key Features

- **Comfortable Cab**: F3000/X3000 sleeper cab
- **Powerful Performance**: 336-550 HP options
- **Efficient Transmission**: FAST manual or automatic
- **Fuel Economy**: Optimized engine technology
- **Reliable Chassis**: MAN technology

## Technical Specifications

| Item | Parameter |
|------|-----------|
| Model | Shacman F3000/X3000 |
| Engine | WEICHAI WP12/WP13/Cummins ISL |
| Power | 336-550 HP |
| Torque | 1600-2500 N·m |
| Transmission | FAST 12-16 speed / Allison |
| Drive Form | 4×2 / 6×4 / 6×6 |
| Fifth Wheel Height | 1,250-1,400mm |
| GCW | Up to 80 tons |

## Standard Configuration

### Cab
- F3000 flat roof or X3000 high roof
- Sleeper berth
- Air suspension seat
- Air conditioning
- Full dashboard

### Chassis
- MAN technology frame
- Hande axles
- Front leaf spring / rear air suspension
- Disc or drum brakes

### Transmission
- FAST 12/16 speed manual
- Double overdrive
- Optional automatic

## Optional Configurations

- Automatic transmission

- Additional fuel tanks
- Fleet management system

## Applications

- Container transport
- Bulk cargo
- Heavy haulage
- Fuel tanker
