# Storage Tank

## Product Overview

The **Storage Tank** is designed for temporary or permanent storage of liquids including fuel, water, and chemicals. Available in various configurations, it can be used as mobile storage or permanent installation.

## Key Features

- **Versatile Usage**: Fuel, water, chemical storage
- **Modular Design**: Easy installation and relocation
- **Durable Construction**: Long service life
- **Various Sizes**: Multiple capacity options
- **Portable Options**: Skid-mounted available

## Technical Specifications

| Item | Parameter |
|------|-----------|
| Capacity | 5,000 - 100,000 Liters |
| Material | Carbon Steel / Stainless Steel |
| Design Pressure | Atmospheric |
| Design Temperature | -20°C to +80°C |
| Coating | Epoxy / Galvanized |

## Standard Configuration

### Tank Design
- Vertical or horizontal
- Diked containment (optional)
- Access manholes
- Inlet/outlet connections

### Safety Features
- Ventilation system
- Level indicators
- Overflow protection
- Fire protection (optional)

## Optional Configurations

- Various materials
- Heating systems
- Insulation
- Level monitoring
- Pump systems

## Applications

- Fuel storage
- Water storage
- Chemical storage
- Emergency storage
- Temporary storage
