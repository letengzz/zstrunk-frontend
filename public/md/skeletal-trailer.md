# Skeletal Trailer

## Product Overview

The **Skeletal Trailer** (also known as skeleton trailer or container chassis) is designed specifically for container transport. Its lightweight frame design maximizes payload capacity while maintaining structural integrity.

## Key Features

- **Ultra-Lightweight**: Optimized steel frame
- **High Payload**: More cargo capacity
- **Container Versatility**: 20ft and 40ft compatible
- **Port-Friendly**: Designed for container yard operations
- **Durable**: High-tensile steel construction

## Standard Configuration

### Frame Design
- High-tensile steel main beam
- Crossmembers with anti-skid holes
- Galvanized or painted finish

### Container Locks
- 4-position twist locks for 20ft container
- 4-position twist locks for 40ft container
- Lock rods and handles

### Brake System
- Air brake system
- Relay valve
- Spring parking brake

### Suspension
- Mechanical suspension standard
- Air suspension optional

## Optional Configurations

- 3-axle configuration
- Air suspension
- ABS system
- Spare wheel carrier
- Different paint options
- Rear loading ramps (optional)

## Applications

- 20ft container transport
- 40ft container transport
- Port container handling
- Intermodal transport
- Container stacking
