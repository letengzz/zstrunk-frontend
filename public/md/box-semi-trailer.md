# Box Semi Trailer

## Product Overview

The **Box Semi Trailer** provides enclosed cargo transportation with complete weather protection. The rigid box body ensures cargo security and protection from environmental elements.

## Key Features

- **Weather Protection**: Complete enclosure
- **Secure Cargo**: Lockable doors
- **Various Sizes**: Multiple length options
- **Durable Construction**: Steel or aluminum body
- **Versatile**: Wide cargo range

## Technical Specifications

| Item | Parameter |
|------|-----------|
| Overall Length | 8,000 - 13,600mm |
| Overall Width | 2,550mm |
| Overall Height | 3,500 - 4,000mm |
| Payload | 30,000 - 40,000 kg |
| Cargo Volume | 55-95 m³ |
| Axle | FUWA 13 tons × 2 or 3 |

## Standard Configuration

### Body
- Steel or aluminum panels
- Thickness: 0.5-0.8mm
- Reinforced frame
- Anti-skid floor

### Rear Doors
- Swing doors or roll-up door
- Locking mechanism
- Sealing strips

### Brake System
- Air brake system
- Spring parking brake
- ABS optional

### Suspension
- Mechanical suspension
- Air suspension optional
