# Removable Gooseneck Trailer

## Product Overview

The **Removable Gooseneck Trailer** (RGN) features a detachable gooseneck that allows front loading of heavy equipment. This design enables easy loading of tracked machinery and other non-self-loading cargo.

## Key Features

- **Front Loading**: Detachable gooseneck
- **Heavy Capacity**: Up to 100 tons
- **Low Deck Height**: Easy loading
- **Hydraulic Ramp**: Powered loading
- **Versatile**: Multiple configurations
